# 翼心Admin

# Build and Run

## Prerequisites

1. make sure you have node.js && npm installed
```
brew install node
```

## Development
1.
- install dependencies
```
npm i cnpm -g
cnpm i pm2 -g
cnpm i
```
2. Start monitoring file changes & developing
```
npm run dev
```
3. Build all locally
```
npm run clean && npm run build
```
